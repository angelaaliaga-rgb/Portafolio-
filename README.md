Portafolio-1- Fernanda Rojas y Angela Aliaga

En este portafolio desarrollaremos un modelo robusto de clasificación capaz de predecir el nivel de Producto Interno Bruto .

El objetivo principar es analizar y utilizar los datos dados para generar un analisis completo en relacion a la variabilidad economica global

Las 3 etapas corresponden a:
Etapa 1: Análisis Descriptivo e Imputación de Datos
Etapa 2:Reducción de Dimensionalidad con PCA
Etapa 3:Implementación de modelos de clasificación,

Las herramientas utilizadas aprendidas durante el curso fueron:

Lenguaje: Python
Bibliotecas: pandas, numpy, wbgapi y scikit-learn (PCA y modelado).
Control de Versiones: Git y GitHub
